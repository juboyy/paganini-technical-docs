Paganini AIOS — Architecture Blueprint

Infraestrutura neuro-simbólica protegida por isolamento Level 3 e sistemas de auto-monitoramento de consistência lógica. O Paganini AIOS não é apenas executável — é durável, auto-aprimorável e auditável.

O que é o Paganini AIOS?​

O Paganini é uma infraestrutura de inteligência, não um framework de agentes. Opera como um AIOS (Agent-Intelligence Operating System) que orquestra agentes especializados através de um pipeline de 19 estágios com governança rigorosa.

Metáfora oficial: Hermes é o motor; Paganini é o carro.

Princípios Fundamentais​

  1. Container Isolation — Isolamento total por tenant. Dados financeiros exigem separação completa.
  2. Human-in-the-Loop — Nenhum deploy em produção sem aprovação humana via Linear.
  3. Knowledge Persistence — Cada tarefa ensina a próxima. Stage 17 é obrigatório.
  4. Durable Agent — Nunca ser instruído duas vezes. Tarefas viram Skills + Cron automaticamente.

Agente Durável (Garry Tan)​

Inspirado pelos princípios de Agentes Duráveis, o Paganini é projetado para automação perpétua:

Conceito → Protótipo → Avaliação → Codificação → Agendamento (Cron) → Monitoramento

Cada tipo de trabalho possui uma e apenas uma Skill proprietária (MECE — Mutually Exclusive, Collectively Exhaustive), garantindo clareza, zero sobreposição e manutenção simplificada.
