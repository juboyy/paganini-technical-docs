Cyber-Security & Hardening (Level 3)

Segurança inspirada nos princípios de Claude Shannon — a segurança de um sistema deve depender da chave, não do segredo do algoritmo. No Paganini, a segurança depende da arquitetura, não da obscuridade.

Modelo de Ameaças​

Sandbox Isolation — Container per Tenant​

Isolamento total via Docker Containers por Tenant. Cada cliente financeiro opera em ambiente completamente isolado.

Regras de Negação (Deny-by-Default)​

CamadaRegraEnforcement
Filesystem~/.ssh bloqueadoBubblewrap bind mount deny
Filesystem~/.aws bloqueadoBubblewrap bind mount deny
Filesystem.env global bloqueadoSeatbelt read deny
NetworkApenas HTTPS (443)iptables egress rules

NetworkDNS restritoAllowlist de resolvers

Seccomp Profile​

{
  "defaultAction": "SCCOMP_ACT_ERRNO",
  "architectures": ["SCMP_ARCH_X86_64"],
  "syscalls": [
    {
      "names": ["read", "write", "open", "close", "stat", "fstat",
                "mmap", "mprotect", "munmap", "brk", "ioctl",
                "access", "pipe", "select", "sched_yield",
                "clone", "execve", "exit", "wait4", "kill",
                "getpid", "socket", "connect", "sendto", "recvfrom"],
      "action": "SCMP_ACT_ALLOW"
    }
  ]
}
Bloqueios Explícitos
  • ptrace — impede debugging/injection entre processos
  • mount/umount — impede escape do container
  • reboot/shutdown — impede DoS no host
  • keyctl — impede acesso ao kernel keyring
  • bpf — impede eBPF exploitation

Egress Filtering​

Allowlist Configuration​

# infra/security/egress-allowlist.conf
# Format: ALLOW <domain> <port> <protocol>

# LLM Providers
ALLOW api.openai.com 443 HTTPS
ALLOW api.anthropic.com 443 HTTPS
ALLOW generativelanguage.googleapis.com 443 HTTPS

# Infrastructure
ALLOW *.supabase.co 443 HTTPS

ALLOW api.github.com 443 HTTPS
ALLOW api.linear.app 443 HTTPS

# Package registries
ALLOW registry.npmjs.org 443 HTTPS
ALLOW pypi.org 443 HTTPS

# DENY everything else (implicit)

Pre-commit Hooks​

# .pre-commit-config.yaml
repos:
  - repo: local
    hooks:
      - id: detect-secrets
        name: Detect Secrets
        entry: python3 scripts/security/detect-secrets.py
        language: python
        types: [python, javascript, typescript]
      - id: detect-pii
        name: Detect PII
        entry: python3 scripts/security/detect-pii.py
        language: python
      - id: verify-gitignore
        name: Verify .gitignore
        entry: python3 scripts/security/verify-gitignore.py
        language: python
      - id: npm-audit
        name: NPM Audit
        entry: npm audit --audit-level=high
        language: node
