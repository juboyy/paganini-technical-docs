The Human Gate: Linear Approval

A branch de produção é 100% blindada. O deploy prod é condicionado à aprovação humana estrita via Linear.

Fluxo de Aprovação​

Dossiê Linear (Template)​

Cada issue de aprovação contém:

SeçãoConteúdo
PR LinkGitHub pull request com diff completo
Specspec.md com gate token e blast radius
ArchitectureDecisões arquiteturais e trade-offs
GitNexus IntelligenceAnálise de impacto — arquivos e funções afetadas
Codex PromptsPrompts usados para geração de código
Cost/TokensTokens consumidos e custo estimado
Demo VideoVídeo mostrando a feature funcionando
Test ReportResultado dos testes E2E e unitários

Poller de Aprovação​

# Cron job: linear-approval-poller (every 30 min)
# Cron ID: 201cff68
# Model: claude-sonnet-4-6
python3 scripts/linear-approval-poller.py

O poller monitora o workspace Vivaldi (VIV) no Linear e detecta issues movidas para "Done" ou "In Review → Done".
